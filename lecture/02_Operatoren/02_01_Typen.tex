		Es ist nicht ganz so einfach...

		\begin{itemize}

		In Python gilt:
		\begin{itemize}
			\item \pythoninline{float} kann nur Zahlen zwischen \pythoninline{-1.7976931348623157e+308} und \pythoninline{1.7976931348623157e+308} speichern
			\item Kleinster positiver \pythoninline{float} ist \pythoninline{2.2250738585072014e-308}
			\item \pythoninline{int} kann in Python beliebig große Zahlen speichern
			\begin{itemize}
				\item Das ist in den meisten Sprachen nicht so!
		\end{itemize}

		\end{itemize}
