\subsection{Text-Slicing \stage3}

Slicing funktioniert auch mit Text. Dabei wird der Text wie eine Liste von
Buchstaben behandelt. \pythoninline{"Hallo Welt!"[:5]} ergibt beispielsweise den
String \pythoninline{"Hallo"}. Findet ihr einen Ausdruck mittels Slicing ein
Palindrom erkennt? \textit{Ein Palindrom ist ein Wort, das vorwärts und rückwärts
    identisch ist}.
